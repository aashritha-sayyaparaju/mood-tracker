Mood Tracker:

A cute calendar-based mood tracker where you log how you're feeling each day with  color-coded moods and notes.

Features:

- Monthly calendar view with navigation
- 5 mood levels (amazing, good, ok, bad, awful) with color coding
- Add optional notes for each day
- Track streak and total logged days
- All data saves automatically in your browser
- Cute pastel design with playful interface

How to Use: 

Pick a mood from the legend, then click any past day on the calendar to log that mood. You can also add an optional note if you want to remember what happened. Watch your streak grow as you log consecutive days.

Files: 

HTML, CSS, JS, localStorage

Live Demo link: 

https://aashritha-sayyaparaju.github.io/mood-tracker

Data:

All mood and note data is stored locally in your browser using local Storage. Your data is private and continues between sessions.
